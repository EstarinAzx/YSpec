YSpec is a YAML-based structured logic language for describing program behavior in a clean, readable format that can be translated into real code.

## What YSpec is for

YSpec is designed to sit between natural language and source code:

```txt
Natural language -> YSpec -> code
```

It is useful for:
- expressing logic clearly before code generation
- keeping structure without raw JSON noise
- documenting behavior in a machine-friendly format
- serving as a higher-level specification layer

## Feature areas

YSpec currently documents:
- functions
- loops and conditionals
- modules and imports
- classes
- type annotations
- exceptions
- async behavior
- pattern matching
- generics
- macros
